Pomodoro Timer

How To Use

🍕 First, click set time

🍕 Enter how long you want to study for

🍕 Next input how long you want to take a break for (Long breaks will be x3 the short break)

🍕 Then click start to begin studying

🍕 If you want the timer to stop, click end

🍕 If you want to change the time, click set time again
